Qual estrutura de dados você usaria?
1. Buscar uma coleção de registros do banco de dados?
- Lista

2. Registro de chamadas de métodos e funções de um complilador?
- Pilha

3. Sistema de senha de banco, hospitais?
- Fila -> Primeiro que entra é o primeiro q sai (FIFO - First in, First Out)

4. Manipular uma coleção de registro únicos?
- Conjuntos (Set)

5. Organizar relacionamentos de redes sociais (twitter, facebook)
- Grafos
